# cutPic：裁剪上下拼接的3张图片
- 双击运行cutPic.exe
- 选择待剪裁的批量图片的路径（允许文件路径中含中文；最终路径中只能含图片文件）
- 选择存放裁剪结果的路径（路径中可含中文）
- 输入3张图的裁剪尺寸
- 点击提交，若输入无空值且路径符合规则，将会在目标路径下生成top、middle、bottom3个文件夹，分别存放上、中、下位置裁剪出的图片结果。
